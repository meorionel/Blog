## 做了什么东西 📦

**PinkLine**

https://github.com/meorionel/PinkLine

这是我在年初的时候写的博客模版, 从我 23 年开始做博客到现在换过很多主题, 我的博客所有的主题都是我自己写的, 从 `hexo` 到 `vitepress`, 学了 `vue` 之后就开始用 `vue` + `MySQL` 写.

只要我学到了什么新的技术, 我就会第一时间用到我的博客上, 但是因为每次我的技术都在进步, 所以每次要改我都会觉得之前写的太混乱了, 就会重构一次(GPT甚至给我颁发了无限重构奖).

而这个博客模版, 是我在了解到 `nuxt` 后学习并开发的, 说实话 `nuxt` 做内容管理真的很爽, 开箱即用的 SSR, 自动导入, 社区生态, 文件路由... 我现在甚至没法好好用 `vue` 了.

上一个博客是使用 `vue` + `MySQL` 写的, 管理起来太抽象了, 现在 **PinkLine** 使用 `nuxt` 官方的 CMS, 一切都变得轻松了, 所有的文章只是一个 .md 文件, `nuxt` 官方也有自己的编辑器, 只需要安装依赖就可以使用.

其实在写这篇文章的前两天我又重构了一次 **PinkLine**, 是因为我想到了一个极其抽象的写法.

我重构了项目结构, 一般来说在 `/pages` 目录下, 一个 .vue 文件就是一个页面, 例如首页就是 `/pages/index.vue`, 文章列表页就是 `/pages/blog/index.vue`.

但是我用了 `nuxt` 的通配路由 `[...slug].vue`, 无论访问的路径是什么, 都会命中这个文件, 那么我该怎么写页面呢, 那就是直接使用 .md 文件, 我只需要在 `[...slug].vue` 文件中写上:

```typescript
const route = useRoute();
const { data: page } = await useAsyncData(route.path, () => queryCollection("page").path(route.path).first());
```

`nuxt/content` 会自己根据当前的页面路径去寻找对应的 .md 文件, 而页面上的自定义组件元素只需要通过 MDC 导入即可, 就像 vitepress 那样.

当我需要一个新的页面时, 我不需要再创建一个新的 .vue 文件, 然后在里面又写一次 `useAsyncData` 去获取对应的 .md 文件, 而是直接写一个 .md 文件就好了.

我不确定其他人是不是都这么做的, 但是这个写法是我自己想到的, 感觉自己应该是天才.

**Orange**

https://github.com/meorionel/orange

暑假第二个月的时候和群友一起开了一个 Minecraft 的服务器, 按照惯例应该写一个游玩文档帮助新玩家更好地适应, 我们看了其他服务器都是用腾讯文档之类的在线工具, 一开始其他人也是说用在线文档, 但是这毕竟涉及到我的专业领域了, 所以我力排众议直接自己写官网, 也是用的 `nuxt/content`, 还用 `nuxt` 自带的服务端写了几个查询服务器插件数据的接口.

**nuxt-oneko**

https://github.com/meorionel/nuxt-oneko

这个是一个 `nuxt` 的插件, 可以在页面上显示一个麦金塔上的小猫, 它会跟着指针移动, 这个原本是一个js插件, 但是我把它做成了 `nuxt` 的插件, 只需要下载, 然后加到 `nuxt.config.ts` 中就可以了.

**Minecraft3DHead**

https://github.com/meorionel/Minecraft3DHead

这个接口用js写的, 可以获取玩家的 Minecraft 皮肤, 然后通过裁切再拼接成 3D 的头颅, 是用在 Orange 文档的接口.

## 买了什么 💸

**MacBook**

一台二手的 MacBook Pro 2017, 在年初的时候花了一千块从海鲜市场淘来的, 圆了我一直想拥有一台 MacBook 的愿望, 我是真的很喜欢苹果的设计, 这台笔记本无论是从内部软件的ui设计还是外部的模具设计, 都深得我心.

但是, 说实话这台笔记本七八年了, 8G 的 RAM 让它几乎干不了什么太重的活.

做一些项目, 刚开始写还好, 写到后面功能多了一开 `dev` 风扇就开始转, 然后内存开始跑到80%, 整台电脑都变卡了, 特别是实训周的时候, 同时开了 idea 启动了一个 spring boot 后端, 然后有开了 trea 跑前端, 直接卡的没边了.

而且因为我几乎每天都会听音乐, 但是这台电脑完全不能支撑我一边开网易云一遍写代码.

但是我并不后悔它, 尽管它隔三差五就会卡, 但我每次写代码都会选择它而不是另外一个台式的 Windows.

**Lumix Camera**

这就是另外一个很多年的愿望了, 高中就想买一台相机拍照, 但是经济实力有限, 只能买一台 ccd, 不过那台 ccd 也是承载了很多青春的记忆, 拍了很多高中时期的照片.

我考上本科, 舅舅奖励了我五千块钱, 本来是让我买笔记本的, 但是因为我已经买了 MBP 了, 索性就圆我另外一个梦想, 买了这台相机, 在京东入手的全新的 松下G85+25mmF1.7套机.

本来我是想着有相机放假可以出门拍拍照, 但结果就是只带出去过两三次, 其他时间都在吃灰.

**Xiaomi Pad**

首发买的 xiaomi pad 8, 8 + 128.

都说买平板都是 "买前生产力, 买后爱奇艺", 还好我买平板就是冲着 "爱奇艺" 去的.

平时都用平板刷视频, 还有玩玩 1999, 还是挺不错的, 除了不方便, 其他的体验要比手机舒服, 也没有出现吃灰的问题.

**Sound**

今年买了俩耳机, 都是水月雨的, 水月雨的忠实粉丝说是.

一个是 菊 另外一个是 太空漫步.

日常在宿舍在家里都喜欢用有线的菊, 菊个头小, 长时间戴也不会有挤压感.

太空漫步就是出门用的, 蓝牙的方便, 但是因为它个头大, 戴久了会感觉耳朵挤着痛

还有一个漫步者的音响, 平时连着电脑用, 但是洗澡的时候我会拿来放各种土味DJ, 要不然会感觉索然无味(我们广西人是这样的).
